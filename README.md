Plant-List-Automated-Labeling-System-V4.1 is a professional Word VBA tool for automatically annotating Chinese plant names with standardized Latin names based on an Excel glossary. It provides first-occurrence-only labeling, smart content auditing, mismatch detection and correction, highlight avoidance, tokenized rendering for layout stability, and botanical formatting rules for scientific nomenclature. The project is designed as a scientific research utility for plant lists, floristic documents, and academic manuscript preparation.
